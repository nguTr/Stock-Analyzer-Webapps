## Download Node.js
https://nodejs.org/en/download

## Install Angular CLI
```
npm install -g @angular/cli
ng new Stock-Analyzer-Webapps
```

## Run
```
npm run start
```
OR
```
ng serve
```

## Visit/Test
```
localhost:4200
```

## Push angular project to GitHub
```
git remote add origin <repository-url>
git remote -v
git branch -M main
git push -u origin main
```
